bezier.js

introduction

bezier.js,is a fast and light-weight implementation of bezier curve in pure JavaScript.

size: ~4kb unminified, ~2kb minified.
performance: near native implementation(or even greater)
controllable: full controllable
usable range: all nth order bezier curves, and even some curves like 6th order the x coords but 8th order the y coords.
transparency: you can see every result of computing, nothing is hidden

inside

Bezier curves can be implemented like parametric plotting, and the plotting function is controlled by the control points. bezier.js does all of this for you, with everything transparent. We compile a curve into two high-performance javascript functions,one is x and the other is y, then call them every time with a specified value t as parameter, and get x value and y value. Then we get these points, and plot them on the canvas.
